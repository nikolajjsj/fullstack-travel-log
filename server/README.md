# The server and API for the App

## Commands:
To run the mongodb database:
- mongod --config /opt/homebrew/etc/mongod.conf

## Things to save in the database

### Log Entry
- Title (Text)
- Description (Text)
- Image (Text)
- Comments (Text)
- Rating (scale of 1 - 10)
- Start Date (DateTime)
- End Date (DateTime)
- Latitude (Number)
- Longitude (Number)
- Created At - (DateTime)
- Updated At - (DateTime)
